PAYCRAVE

This project is a mobile application intended for users searching for food trucks in their area. It helps customers choose and pay as well as provide them with food truck ratings, reviews and proximity information. The task was to promote ease, clarity, minimal interaction and security in the payment

process. The development procedure is similar to a later project although this enabled me to have the opportunity to create my first design mockup. The color palette, fonts and logo were provided for this project while I presented user stories, user flow, wireframes and color design mockup.
