ALEKS & MILO

This is a ficticious ecommerce site Bloc project intended for fashion-concious shoppers. It’s task is to entice buyers with alluring images, simple navigation, easy and secure payment process and a search and filter options for users in search of a specific size, choice, and option.

My design plan was to create a very simple layout structure spotlighting the images and feel of the products and impemented a mobile version UX design. I made a straight-forward sitemap, basic wireframes, a colored web designed pages and an iOS version of the site.
