ALEKS & MILO

This Bloc project is a fictitious ecommerce site intended for sophisticated fashion-conscious shoppers. Its task is to attract buyers with hot and trendsetting images, simple navigation as well as an easy and secure payment process. Search and filter options are also provided for users with

more specific needs. My design plan was to create a very simple layout structure highlighting the chic appearance and feel of the products along with an implemented mobile version UX design. I made a sitemap, user stories, basic wireframes, colored web designed pages and an iOS version of the site.
