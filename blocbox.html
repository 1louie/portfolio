BLOCBOX

Blocbox was the introductory project of my apprenticeship for BLOC. The lesson was to follow the process through guided tutorials. These were broken up into groups using planning, structure, layout and visual. The flow of the project consisted of creating user personas, user stories, user flows,

a sitemap, low-fidelity wireframes, high-fidelity mockups, an interactive prototype, HTML/CSS development and Usability Testing. The planning process involved conducting a survey with friends using Google Drive so I can analyze their responses and learn the specifics of each person’s needs.
